Colocar la imagen adjunta (Interstellar.png) dentro del div con clase "fondo", de tal manera que la imagen ocupe toda la pantalla sin repetirse.(Ver Resultado.png)


Pasos:

Aplicarle a clase "fondo" :

  1)-. La propiedad background para cargar la imagen.
  2)-. La propiedad background-repeat para que no se repita la imagen.
  3)-. La propiedad  background-position para que se centre la imagen.
  4)-. Se coloca la propiedad height y width con un 100% para que ocupe todo el contenedor.
  5)-. Se coloca la propiedad background-size para que la imagen cubra todo el div.


Aplicarle a las etiquetas html y body :

  1)-. Se coloca la propiedad height y width con un 100% para que ocupe toda la pantalla.
